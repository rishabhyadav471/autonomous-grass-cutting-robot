
# Components Used

## Microcontroller
- Arduino UNO

## Sensors
- Ultrasonic Sensor (HC-SR04)

## Actuators
- DC Motors (for robot movement)
- DC Motor (for grass cutting blade)

## Motor Control
- L298N Motor Driver Module
- Relay Module (for cutter motor control)

## Power Supply
- Battery (9V / 12V)

## Mechanical Parts
- Robot Chassis
- Wheels
- Grass Cutting Blade
- Motor Mounts

## Miscellaneous
- Jumper Wires
- On/Off Switch
- Connecting Cables
